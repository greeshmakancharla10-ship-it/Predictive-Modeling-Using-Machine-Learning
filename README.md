# Predictive Modeling Using Machine Learning

## Project Overview

This project focuses on building machine learning models to predict student performance using the Student Performance Dataset. The project demonstrates the complete machine learning workflow, including data preprocessing, feature engineering, model training, evaluation, and visualization of results.

## Objective

The objective of this project is to predict whether a student is a high performer or low performer based on demographic and educational factors such as gender, parental education level, lunch type, and test preparation course.

## Dataset

The project uses the Student Performance Dataset, which contains information about:

* Gender
* Race/Ethnicity
* Parental Level of Education
* Lunch Type
* Test Preparation Course
* Math Score
* Reading Score
* Writing Score

## Machine Learning Algorithms

### Decision Tree Classifier

Used to classify students into performance categories based on input features.

### Random Forest Classifier

An ensemble learning method used to improve prediction accuracy and model reliability.

## Evaluation Metrics

The models were evaluated using:

* Accuracy Score
* Confusion Matrix
* Classification Report
* Feature Importance Analysis

## Key Findings

* Test preparation course completion positively impacts student performance.
* Parental education level influences academic achievement.
* Random Forest provides more robust predictions compared to a single Decision Tree.
* Educational and demographic factors can be used to predict student performance effectively.
